## 功能特色

- 📝 **互動式作答** — 逐題作答，即時顯示答案與解析
- 🔍 **多元篩選** — 依年度、題型、難度、學習概念選題
- 📊 **成績統計** — 自動計算正確率與錯題複習
- ⚡ **純靜態網頁** — 無需伺服器，GitHub Pages 托管

## 題庫規格

| 項目 | 內容 |
|------|------|
| 總題數 | 406 題 |
| 涵蓋年度 | 102–114 年（民國）|
| 有專家答案 | 386 題 |
| 題型 | vocabulary、grammar、reading、cloze、conversation |
| 難度分級 | basic、medium、difficult |
| 學習概念 | 100+ 種（如 present simple、passive voice 等）|

## 使用方式

1. 開啟 [題庫網站](https://killkli.github.io/tw-exam-bank)
2. 點擊「依條件選題」或點擊年份卡
3. 作答並點「確認答案」
4. 完成後查看成績與錯題複習

## 題庫來源

台灣國中教育會考歷屆試卷（[cap.rcpet.edu.tw](https://cap.rcpet.edu.tw/examination.html)），由 Vision AI（Qwen）視覺辨識轉換為 JSON 格式，經三階段 QA 系統（研究者→專家→評估）補正。
